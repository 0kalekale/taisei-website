Taisei (lit. Occident) is a fan-made Open Source clone of the Touhou series, written in C using SDL/OpenGL/OpenAL.

Basically, it is a shoot ’em up game with lots of bullets (the genre is literally called bullet hell). The game is mostly done and very playable, so feel free to give it a try!

(Important: Version 1.0a is outdated and will soon be replaced by new, much better release.)
